# FW V1

Framework Version 1 - Trading System Framework

## Overview

This repository contains the framework for algorithmic trading systems.

## Quick Start

```bash
# Clone
git clone https://github.com/yujin7/FW-V1.git

# Setup
cd FW-V1
pip install -r requirements.txt
```
